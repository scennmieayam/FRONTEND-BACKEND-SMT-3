## Yang Ada di Sini

- `index.html` - File HTML dengan berbagai contoh select

- Selected options

- Select biasa
- Multiple select
- Select dengan groups
- Select dengan disabled option

## Teknologi

- HTML
- JavaScript (sedikit)
